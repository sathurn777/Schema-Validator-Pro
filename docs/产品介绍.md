# Schema Validator Pro — 产品介绍

## 📌 产品定位

**Schema Validator Pro** 是一个生产级的 Schema.org 结构化数据生成与验证工具，专为 WordPress 站点和内容管理系统设计。

## 🎯 核心功能

### 1. 智能 Schema 生成
- **9 种 Schema 类型支持**：Article（文章）、Product（产品）、Recipe（食谱）、Event（活动）、Organization（组织）、Person（人物）、FAQPage（常见问题）、HowTo（教程）、Course（课程）
- **自动内容分析**：根据输入内容智能生成符合 Schema.org 标准的结构化数据
- **嵌套对象支持**：所有嵌套对象（如 author、publisher、offers）都带有正确的 `@type` 标记
- **字段规范化**：自动规范化日期（ISO8601）、URL（绝对路径）、货币（ISO4217）、语言（BCP47）

### 2. 严格 Schema 验证
- **多层验证机制**：
  - 必需字段检查（基于 Schema.org 规范）
  - 推荐字段检查（SEO 最佳实践）
  - 字段类型验证（包括嵌套对象）
  - 字段值验证（URL、日期、枚举等）
- **质量评分系统**：0-100 分评分，清晰展示 Schema 质量
- **优化建议**：针对每种 Schema 类型提供具体的优化建议

### 3. WordPress 无缝集成
- **可视化插件界面**：在 WordPress 编辑器中一键生成和发布 Schema
- **REST API v2 集成**：通过 WordPress REST API 自动注入 Schema 到文章
- **Application Password 认证**：安全的身份验证机制

### 4. 生产级监控
- **Sentry 错误追踪**：自动捕获和报告错误
- **Prometheus 指标收集**：实时性能监控和业务指标
- **Structlog 结构化日志**：便于分析和调试的 JSON 格式日志

## 🚀 快速开始

### PyPI 安装
```bash
pip install schema-validator-pro
```

### CLI 使用
```bash
# 启动服务
schema-validator-pro --host 0.0.0.0 --port 8000

# 查看版本
schema-validator-pro --version

# 查看帮助
schema-validator-pro --help
```

### Python API 使用
```python
from backend.services import SchemaGenerator, SchemaValidator

# 生成 Schema
generator = SchemaGenerator()
schema = generator.generate(
    schema_type="Article",
    content="My Article Title\n\nArticle content...",
    url="https://example.com/article"
)

# 验证 Schema
validator = SchemaValidator()
result = validator.validate(schema)
print(f"Quality Score: {result['score']}/100")
```
