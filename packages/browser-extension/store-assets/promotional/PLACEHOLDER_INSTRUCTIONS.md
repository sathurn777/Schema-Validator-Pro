## 📐 快速创建模板

### 使用 Canva 创建（推荐）

1. 访问 Canva: https://www.canva.com/
2. 搜索 "Chrome Extension Promo"
3. 选择合适的模板
4. 自定义内容：
   - 替换文字为 "Schema Validator Pro"
   - 上传扩展图标
   - 调整配色为品牌色
5. 导出为 PNG（高质量）

### 使用 Figma 创建

1. 创建新文件
2. 设置画布尺寸：
   - 小型: 440x280
   - 大型: 920x680
   - 侯爵: 1400x560
3. 添加背景色
4. 导入扩展图标
5. 添加文字和装饰元素
6. 导出为 PNG (2x)

## 🎯 设计要点

### DO ✅
- 使用高对比度，确保文字清晰
- 保持设计简洁，避免过度装饰
- 突出扩展的核心功能
- 使用品牌配色
- 确保图片质量高清

### DON'T ❌
- 不要使用模糊或低质量的图片
- 不要包含误导性信息
- 不要使用受版权保护的图片
- 不要使用过小的字体（< 12px）
- 不要过度使用动画或特效

## 💡 快速占位符

如果你只是想测试发布流程，可以使用简单的占位符：

### 使用在线工具
访问: https://placehold.co/440x280/3B82F6/FFFFFF/png?text=Schema+Validator+Pro

### 使用 ImageMagick
```bash
# 小型宣传图
convert -size 440x280 xc:'#3B82F6' \
  -gravity center \
  -pointsize 32 \
  -fill white \
  -annotate +0-20 "Schema Validator Pro" \
  -pointsize 18 \
  -annotate +0+20 "Validate & Generate Schema.org JSON-LD" \
  promo-small-440x280.png

# 大型宣传图
convert -size 920x680 xc:'#3B82F6' \
  -gravity center \
  -pointsize 48 \
  -fill white \
  -annotate +0-40 "Schema Validator Pro" \
  -pointsize 24 \
  -annotate +0+20 "Professional Schema.org Validator" \
  promo-large-920x680.png
```

---

**注意**: 
- 小型宣传图 (440x280) 是**必需**的，没有它无法发布到 Chrome Web Store
- 大型和侯爵宣传图是可选的，但强烈推荐创建以提高扩展的可见度
- 所有宣传图片仅用于商店展示，不会包含在扩展包中
